# NBA Clutch Performance Dashboard

Analyzing which NBA players rise (or shrink) in clutch moments.

**Season:** 2024-25  
**Data:** NBA Stats API via nba_api  
**Tools:** Python · Pandas · Plotly · Seaborn

## How to run
1. `pip install -r requirements.txt`
2. Open `nba_clutch_dashboard.ipynb` in Jupyter or VS Code
